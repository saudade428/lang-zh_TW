Hide Deprecated Applications=隱藏已棄用的應用
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=此設置將在可用應用列表中隱藏已棄用的應用程式, 但已棄用的應用仍將顯示在已安裝的應用和以前的應用列表中
Hide Incompatible Applications=隱藏不相容的應用
This setting will hide applications that are listed as being incompatible with your version of Unraid=此設置將隱藏與您的 Unraid 版本不相容的應用程式
Allow install of second instance=允許安裝第二個應用
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=此選項會在卡片和彈出窗口上啟用一個新按鈕,該按鈕允許您使用默認模板值重新安裝已安裝的應用程式。此選項有時可能會造成混淆，通常僅在您希望安裝並同時執行兩個相同的應用程式時才使用。.
Automatically open the sidebar=自動打開側邊欄
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=滑鼠懸停在應用程式圖示上時, 允許社區應用中心自動打開側邊欄以顯示更多資訊
Allow CA to check for updates to applications=允許社區應用中心檢查應用程式的更新
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=此設置使社區應用中心能夠檢查應用程式的更新, 請注意：社區應用中心將始終檢查自身的更新
Disable Mouse Wheel Page Changes=關閉滑鼠滾輪翻頁
This setting disables using left or right on a mouse wheel to change the page=此選項用於關閉滑鼠左/右方向滾輪翻頁
Mouse wheel threshold=滑鼠滾輪閾值
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=當你將滑鼠滾輪向左或向右移動時，瀏覽器會返回相應的變化值。請將這個變化值設置為閾值。需要注意的是，不同的瀏覽器和操作系統可能會返回不同的數值。
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=這是一個對所有設備都適用的全局設置。如果出現問題，請禁用滑鼠滾輪翻頁功能。
Allow CA to send any emergency notifications=允許社區應用中心發送緊急通知
This setting allows CA to periodically scan your system and alert you to the presence of=此設置允許社區應用中心定期掃描您的系統，並向您提醒系統中存在的問題
1 Known malware and security vulnerabilities within applications=1 應用程式中已知的惡意軟體和安全漏洞
2 Critical security vulnerabilities within Unraid OS=2 Unraid OS 系統中的嚴重安全漏洞
3 Any installed applications which may seriously hinder the operation of your server=3 任何可能嚴重阻礙伺服器運行的已安裝的應用程式
Save CA debugging information=保存社區應用中心的除錯資訊
Only enable this when advised to from the author of Community Applications=僅當社區應用程式的開發者建議時才啟用此功能
Enable developer mode=啟用開發者模式
This unlocks some extra features within CA generally only used by developers Not recommended to enable=這會解鎖社區應用中心中的一些額外功能(通常僅由開發人員使用, 不建議啟用)
